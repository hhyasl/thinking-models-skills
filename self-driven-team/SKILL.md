# 自驱型团队

## 简介

> **"你的团队文化 = 你的团队行动"**

## 核心模型体系

### 一、核心公式与三大规则

#### 文化公式

```
你的团队文化 = 你的团队行动
```

文化不是与生俱来的天赋，而是一种可以训练和培养的能力。

#### 三大规则

| # | 规则 | 核心 |
|---|------|------|
| ① | **从当前现实开始** | 别幻想乌托邦，从团队的优势和劣势出发 |
| ② | **建立对话而非颁发指令** | 每个人都为文化建设出力，共同探索 |
| ③ | **不设定规则** | 视作可测试的操作集合，而非僵化蓝图 |

### 三、安全感支柱（铸造归属感）

#### 3.1 归属感暗示（核心机制）

安全感通过微小而鲜明的行为传递的信号：

```
· 我们心连着心
· 我们拥有共同的未来
· 我在乎你
· 你在这里拥有话语权
· 你很重要
```

大脑收到这些信号时，从"警惕模式"切换到"联结模式"。

#### 3.2 PALS入职模型

| 阶段 | 含义 | 关键动作 |
|------|------|---------|
| **P** Prepare | 入职前准备 | 邀请新人加入群聊，提前发送欢迎礼包 |
| **A** Arrive | 第一天迎接 | 安排迎接午餐，第一次午餐禁止聊工作 |
| **L** Launch | 融入工作 | 向新人展示流程，安排导师 |
| **S** Support | 持续支持 | 使用问题清单了解新成员，定期Check-in |

#### 3.3 零容忍"聪明的混蛋"

> 没有任何人可以凌驾于团队之上，无论他多么才华横溢。
>
> "聪明混蛋"的收益几乎永远无法弥补给团队造成的绩效损失。在重视尊重的团队中，人们相互分享信息的概率高出**59%**。

**3种应对方法：** ①制定明确的行为准则 ②及时一对一沟通 ③屡教不改则坚决清除

#### 3.4 浅显乐趣 vs 深度乐趣

| 类型 | 定义 | 效果 |
|------|------|------|
| 🍬 浅显乐趣 | 一起做愉快事（玩游戏、听音乐） | 像肾上腺素，短暂提升 |
| 🏔️ **深度乐趣** | 团队共同主导体验，自己负责 | **利润是浅显乐趣的4倍** |

**如何创造深度乐趣：** 让团队设计自己的工作场所、组织外出活动、自主决定奖品分配。

#### 3.5 个人最佳状态文档（我的使用说明书）

每人1页纸描述：
```
· 我状态最佳的时候是________
· 我状态最差的时候是________
· 他人可以依靠我做________
· 我需要他人帮我的是________
```

> 了解同事的"电量表"比了解KPI更重要。

**团队章程版本（团队最佳状态文档）：** 操作规范、责任分工、核心价值观、沟通方法、会议节奏。

#### 3.6 能量检查活动

会议开场时，每人给自己能量水平打分（1~7分），然后做一个改变环境的举措提分。

### 四、脆弱性支柱（淬炼坦诚）

#### 4.1 行动后复盘（AAR）

海豹突击队改良，完成重要任务后围绕3个问题：

```
· 哪些方面进展顺利？
· 哪些方面进展不顺利？
· 下次我们将如何改进？
```

**核心原则：** 不是为了表扬或指责，而是共同学习。不受职级影响，每个人都保持谦逊。

> 每天花15分钟反思的团队，技能测试成绩比对照组高**20%**。

**行动前评估（补充）：**
- 我们预期的结果是什么？
- 我们可以预见哪些挑战？
- 我们或他人从类似情形中学到了什么？

#### 4.2 3行邮件重建信任

给团队发送包含3个问题的邮件：
1. 我坚持做的、你希望我继续做的是什么事？
2. 我做得不够、你认为我应多做的是什么？
3. 为了让你更有效率，我能做什么？

**进阶4问（Humu公司）：**
- 我做的哪件事对团队贡献最大？
- 为了改善关系，你有哪些事需要我知道？
- 你的哪种天赋被忽视或未充分利用？
- 你的驱动力是什么？

#### 4.3 倾听的神奇短语："再讲讲"

> 这3个字极为有用，因为问题像冰山——暴露表面的不起眼，隐藏的却庞大复杂。

**深度倾听3不原则：** 不打断、不评判、不急着给方案

**善于倾听者的4种行为（曾格和福克曼研究）：**
1. 以令人感到安全和支持的方式互动
2. 愿意帮助他人，也愿意合作
3. 以温和建设性的方式挑战固有观念
4. 偶尔提出建议，帮他人打开思路

#### 4.4 反馈变形记

将评判性反馈改为引导反思的问题：

| 不要（评判） | 要（引导反思） |
|-------------|--------------|
| "这方案不专业" | "如果对标行业顶尖，可以增加哪些要素？" |
| "你做得不对" | "你认为哪些方面可以做得更多？" |
| "这里有问题" | "你认为哪些工作可以用不同方式完成？" |

#### 4.5 面对面谈负面信息

> 越是想逃避的对话，越值得当面说清。

**三段式表达：** 我观察到的事实 → 我的感受 → 我希望改进的方式

### 五、使命感支柱（点燃意义）

#### 5.1 标语 / 口号地图

团队共同创建"真北"地图，明确：
- 团队的核心方向（真北）
- 需要做出的关键行为
- 反复面对的问题与理想应对
- 绝不应出现的行为

**制作步骤：**
1. 分享"真北"故事和标语地图概念
2. 每组设计自己的地图：方向？关键行为？标语？禁止行为？
3. 分享结论，激发对话
4. 融入日常工作，随挑战变化而迭代

#### 5.2 优势/障碍研讨会

约1小时，2个问题：
1. **我们团队的优势在哪里？** — 举例描述具体行为和沟通方式
2. **什么障碍阻止我们保持优势？** — 越具体越好

**进阶：** 将"优势/障碍"配对写白板（如"执行力强→容易忽略他人意见"），为每个配对写一句"平衡建议"。

#### 5.3 赋能式开会

**1. 聊好奇心：** 会议中途暂停，问"你对什么事情感到好奇？"给20秒思考，每人口头分享。
**2. 热身破冰（远程）：**
  - 15秒闪电回顾（两句话描述近况）
  - 能量检查（打分+改善举措）
  - 音乐分享活动
  - 深呼吸活动

#### 5.4 唤醒使命感仪式

每次开会前用真实故事连接工作与宏大目标。

> 一个真实的困境故事，比全年战略宣讲更能燃起团队斗志。

**建立"意义制造机"机制：** 共享空间收集客户感谢信、每周分享"使命故事"、将核心客户故事转化为视觉提醒。

#### 5.5 断舍离日（减法游戏）

匿名列出"最想废除的制度/流程"→ 统计得票率 → 选1~2项"停用两周"实验 → 为废除的制度写墓志铭。

## 典型应用场景

### 场景一：团队缺乏凝聚力，成员各自为战
用**归属感暗示**检查→用**PALS模型**排查新成员融入→用**行动后复盘（AAR）** 建立共同学习习惯→用**深度乐趣**创造共同体验。

### 场景二：新员工来了留不住
用**PALS入职模型**逐项检视入职流程→用**个人最佳状态文档**加快相互了解→用**零容忍混蛋**确保团队环境健康。

### 场景三：团队氛围压抑，不敢说真话
用**脆弱性支柱**——领导者先用**3行邮件**示范坦诚→用**面对面谈负面信息**规则改变沟通方式→用**反馈变形记**训练团队给反馈。

### 场景四：会议低效，目标感模糊
用**能量检查**开场→用**聊好奇心**打破僵局→用**唤醒使命感仪式**连接工作意义→用**AAR**收尾复盘。

### 场景五：远程团队沟通疏离
用**先面对面工作**规则安排线下见面→用**热身破冰活动**活跃线上会议→用**个人最佳状态文档**加深相互理解→用**"再讲讲"**培养深度倾听。
