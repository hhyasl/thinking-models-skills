# AI学习高手

## 简介

> **"AI是登山路上的拐杖，但决定攀登哪座山的，是你自己。"**

本技能提供一整套AI时代的学习方法论，覆盖从**制定目标→高效学习→应对考试→深度思考→人际沟通→向AI提问**的完整学习闭环。每个方法都配有可直接使用的AI提示词。

### 二、目标力

#### 目标制定4步法

| 步骤 | 操作 | 避免 |
|------|------|------|
| ① 明确愿望 | 写下模糊的想法 | "我要学好数学" |
| ② SMART评估 | 用AI评估目标有效性 | Specific/Measurable/Achievable/Relevant/Time-bound |
| ③ AI引导细化 | AI用教练技术引导制定目标 | 从"每天读20页"变成"每天早起朗读4句古诗" |
| ④ 行动拆解 | 拆解为可执行事项 | 可执行、可检查、可坚持 |

**提示词模板：**
```
【困境】我是一名XX年级学生，想提高XX，给自己制定了"XX"的目标。
【需求】请你先按照SMART原则评估我目标的有效性，再用教练技术一步一步引导我制定适合我且符合SMART原则的目标。
【限制】你可以要求我提供所需的个人相关信息。
```

### 三、学习力

#### 3.1 AI版费曼学习法（6步）

| 步骤 | 操作 | AI的角色 |
|------|------|---------|
| ① 选择知识点 | 告诉AI要学的主题 | 接收 |
| ② 用自己的语言解释 | 像教小白一样讲给AI听 | 认真听 |
| ③ 识别漏洞 | — | 精准指出理解盲区 |
| ④ 回顾和补充 | 回到课本重新学习 | 引导回到原始材料 |
| ⑤ 简化表达 | 尝试更简单的说法 | 帮你压缩和类比 |
| ⑥ 总结和应用 | 出题检验掌握程度 | 出题测试+批改讲解 |

**提示词模板：**
```
【角色】你是一位耐心的导师，擅长用费曼学习法引导我学习。
【任务】通过费曼学习法帮助我掌握知识点。
【引导步骤】1.选择知识点→2.用自己的语言解释→3.识别漏洞→4.回顾和补充→5.简化表达→6.总结和应用
```

#### 3.2 西蒙学习法（知识拆分）

**4步骤：**
1. **确定核心目标** — 明确核心问题
2. **拆分知识模块** — 拆分为可独立学习的模块
3. **刻意练习** — 每个模块出题逐个击破
4. **总结答案** — 整合理解形成完整思路

**每个模块包含：** ①知识点（概念/公式/定理） ②常见错误与纠错

#### 3.3 艾宾浩斯记忆复习表

**科学复习间隔：** 第1天 → 第2天 → 第4天 → 第7天 → 第15天 → 第30天 → 第60天

**提取练习原理：** 不要重读笔记！遮住答案回忆，提取越费力记忆越深刻。

> 用提取练习的学生成绩比重读笔记的高**17%~26%**；压力增大时重读组下降32%，提取组几乎不受影响。

**提示词模板：**
```
【角色】你是一位XX学科老师。
【技能】能够理解艾宾浩斯遗忘曲线原理和提取练习原理。
【任务】为我设计一份基于艾宾浩斯遗忘曲线及提取练习原理的记忆表，主题为"XX"。
【任务要求】设定合理的复习时间间隔；设计包含复习周期、日期、内容与方法、时长的表格。
```

### 四、考试力

#### 4.1 分级考核法

AI出题采用**难度分级递进**：

| 层级 | 规则 |
|------|------|
| 容易 | 简单选择题开始，一次一道 |
| 中等 | 答对2次 → 升级难度 |
| 较难 | 答错 → AI讲解+同知识点再出一道 |
| 挑战 | 持续升级直至完全掌握 |

**提示词模板：**
```
【任务】你是一位资深的XX老师，需要引导我进行一场关于"XX"知识的问答挑战。
【工作流程】1.从最简单的选择题开始，每次一道。2.答对2次升级难度，答错讲解+再出。3.不要一次性出多道题。
```

#### 4.2 清单学习法

把知识点/考点整理为检查清单，逐项核对掌握程度。

**提示词模板：** "请将XX章节的知识点整理成一份学习检查清单，包含'已掌握/需巩固/未掌握'三栏。"

### 五、思考力

#### 5.1 批判性阅读框架

阅读时不问"作者说了什么"，而是问：
- 作者的核心论点是什么？
- 论据是否充分、可靠？
- 有没有相反的视角或证据？
- 我同意哪些、不同意哪些？为什么？

#### 5.2 跨界思维

"如果经济学家怎么看？物理学家怎么看？艺术家怎么看？"

**提示词模板：** "请从数学家、历史学家、心理学家的角度分别分析'XX'这个问题。"

### 六、沟通力

#### 6.1 师生沟通三步法

| 步骤 | 话术示例 |
|------|---------|
| ① **描述事实，表达感谢** | "老师，上次您给我讲那道题后我豁然开朗，谢谢您" |
| ② **说明目标，阐述困难** | "我想把数学提到90分，但二次函数这块总卡住" |
| ③ **给出策略，邀请配合** | "我计划每天做2道函数题，您方便每周帮我批改一次吗？" |

#### 6.2 情绪价值框架

> **每种情绪都有价值，情绪越强烈，包含的信息越重要。**

| 情绪 | 它在告诉你什么 |
|------|--------------|
| 😨 恐惧 | 有危险需要警觉 |
| 😢 悲伤 | 失去了重要的东西 |
| 😰 紧张 | 这件事很重要，需要专注 |
| 😡 愤怒 | 边界被侵犯了 |
| 🤢 厌恶 | 需要远离某些东西 |

#### 6.3 鳄鱼定律（冲突处理）

被沼泽中的鳄鱼咬住腿时不挣扎 → 果断止损 → 活下来。处理人际冲突同理——**及时抽离比"争个输赢"更重要。**

### 七、提问力

#### AI高效提问三段式框架

| 要素 | 内容 | 示例 |
|------|------|------|
| **【角色】** | 告诉AI你是谁/想让AI扮演谁 | "你是一位耐心的小学数学老师" |
| **【任务/需求】** | 告诉AI需要它用什么方法辅助你 | "请用费曼学习法引导我掌握这个知识点" |
| **【限制/要求】** | 让AI用特定方式辅助 | "不要一次性出多道题，一次只出一道" |

#### 六种学习场景提示词速查

| 场景 | 核心提示词模板 |
|------|--------------|
| 🎯 制定目标 | "请按SMART原则评估我的目标，用教练技术引导我制定" |
| 📖 费曼学习 | "请用费曼学习法6步骤引导我掌握XX知识点" |
| 🔧 西蒙拆分 | "请按西蒙学习法将这道题拆分为知识模块" |
| 📝 分级考核 | "从简单题开始，答对2次升级难度，答错讲解+再出" |
| 🗓️ 复习计划 | "请基于艾宾浩斯遗忘曲线设计复习表，包含时间间隔和复习方法" |
| 💬 沟通练习 | "请模拟与老师的对话，帮我练习沟通三步法" |

## 典型应用场景

### 场景一：学不进去，不知道怎么开始
用**目标制定4步法**把模糊的愿望变成可执行的计划 → 用**SMART+AI**细化每日行动 → 用**西蒙学习法**把大任务拆成小模块。

### 场景二：上课听懂了，做题还是不会
用**AI版费曼学习法**把知识点讲给AI听 → AI指出你的"假懂"盲区 → 用**分级考核法**逐级测试直到真正掌握。

### 场景三：复习没效率，记了就忘
用**艾宾浩斯复习表**制定科学复习计划 → 用**提取练习**替代重读笔记 → 用**清单学习法**逐项核对。

### 场景四：不知道怎么跟老师沟通
用**师生沟通三步法**组织语言 → 用AI模拟对话练习 → 实际使用三步法开口。

### 场景五：学习压力大，情绪不好
用**情绪价值框架**识别情绪背后的信息 → 用**鳄鱼定律**处理冲突 → 接纳情绪而非压抑。
