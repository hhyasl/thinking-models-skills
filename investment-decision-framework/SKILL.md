# 投资决策框架

这是从《AI时代的个人成长·投资·财富自由》一书中提炼的系统化投资决策体系，从宏观趋势分析到微观公司甄选，再到资产配置与心法校准。

## 适用场景

- 用户需要分析某个行业或公司
- 用户想在当前市场环境下制定投资策略
- 用户想做个人资产配置或再平衡
- 用户对某只股票或基金做买入/卖出决策
- 用户需要制定一份可执行的交易计划
- 用户想复盘过去的投资决策

## 核心模型

### 1️⃣ 三层漏斗分析框架

投资决策的核心引擎，像精密滤网逐层聚焦。

```
┌─────────────────────────────────────┐
│  第1层：宏观视野漏斗               │
│  · 人口结构、技术革命、地缘格局    │
│  · 问：潮水流向哪里？              │
├─────────────────────────────────────┤
│  第2层：中观赛道漏斗               │
│  · 产业政策、行业周期、竞争格局    │
│  · 问：哪个赛道有长期坡道？        │
├─────────────────────────────────────┤
│  第3层：微观公司漏斗               │
│  · 公司治理、财务健康、护城河      │
│  · 问：谁值得长期持有？            │
└─────────────────────────────────────┘
```

**使用流程：**
1. **自上而下导航**：先扫宏观，再定行业，最后选公司
2. **自下而上反馈**：微观异常反过来修正中观/宏观判断
3. **AI加速**：用AI收集各层数据、制作趋势图表、对比分析

**典型提问：** "请用三层漏斗框架帮我分析新能源车行业，先看宏观趋势，再看产业格局，最后筛选出值得关注的龙头公司。"

### 2️⃣ "慢·中·快"三变量框架

区分变量时间尺度，避免用短期噪音做出长期决策。

| 变量 | 时间尺度 | 比喻 | 典型的决策角色 | 典型代表 |
|------|---------|------|--------------|---------|
| **慢变量** | 5~10年+ | 塑造"地形" | 定方向的罗盘 | 人口结构、技术革命、地缘政治 |
| **中变量** | 1~5年 | 制造"季节" | 选行业的导航仪 | 产业政策、信贷周期、供需格局 |
| **快变量** | 1年内 | 产生"天气" | 选时点的温度计 | 季度财报、突发事件、市场情绪 |

**核心判断原则：** 快变量不改变基于慢变量和中变量得出的核心价值判断。

**典型提问：** "请用慢中快三变量框架分析A股当前的配置环境，告诉我哪些是决定长期趋势的慢变量，哪些是短期噪音的快变量。"

### 3️⃣ 投资五步法（修身齐家准备）

在打开交易软件之前，先完成这五步：

1. **本末之辨**：建立投资负面清单——不破坏家庭、不影响主业、不带来持续负面情绪、不用杠杆
2. **情绪管理**：识别命名恐惧与贪婪 + 系统化对冲 + 记录情绪曲线
3. **无为而治**：接受市场90%是"垃圾时间"，战略性精力管控
4. **齐家**：投资纳入家庭整体财务与情感框架，与家人达成共识
5. **决策清单**：最终产出必须是可执行、不可篡改的白纸黑字清单

### 4️⃣ 投资决策清单

将分析结论转化为白纸黑字、条款清晰的"契约"，是情绪的压舱石、纪律的化身。

**标准清单要素：**
- 当前市场环境概述（用慢中快三变量描述）
- 核心投资假设（3~5条关键前提）
- 行动指令（买入/卖出/持有，具体价格区间和仓位比例）
- 仓位约束（单支标的仓位上限，总仓位纪律）
- 止损条件（触发什么条件必须止损，具体数值）
- 复盘指标（何时、以什么标准复盘本次决策）

**典型提问：** "我打算买入腾讯，请帮我生成一份完整的投资决策清单，包含假设、仓位约束和止损条件。"

### 5️⃣ 基于生命周期的战略配置模板

| 阶段 | 年龄段 | 类型 | 权益占比 | 核心目标 |
|------|--------|------|---------|---------|
| 奋斗筑基期 | 25~35岁 | 进取型 | 70%~80% | 最大化长期资本增值 |
| 稳健发展期 | 35~50岁 | 平衡型 | 50%~65% | 增长与风险平衡 |
| 守成释放期 | 50岁+ | 保守型 | 20%~40% | 保值为先、现金流 |

**使用方式：** 作为起点模板，根据个人风险偏好、收入稳定性、已有资产进行微调。

**典型提问：** "我30岁，年收入30万，风险承受能力中等，请帮我制定一份基于生命周期的资产配置方案。"

### 6️⃣ 投资武器三维认知框架

从三个维度审视任何投资工具，避免盲目投资。

| 维度 | 核心问题 | 评估参数 |
|------|---------|---------|
| **① 资产类别** | 你投资的是什么？ | 收益潜力 × 风险等级 × 流动性 |
| **② 分析方法** | 你靠什么判断价值？ | 基本面分析 / 技术面分析 / 量化分析 / 行为金融 |
| **③ 适用场景** | 它在你的组合中扮演什么角色？ | 进攻（权益） / 防御（固收/现金） / 对冲/保障 |

**典型提问：** "请用三维认知框架帮我分析可转债这个投资工具，说明它的资产类别本质、适用分析方法以及适合在组合中扮演什么角色。"

### 7️⃣ "守中"智慧

西方理性框架在A股市场的"本地化编译器"。

**核心要义：**
- 坚守价值内核（慢变量决定长期方向）
- 善用A股生态（政策驱动周期 + 散户情绪带来的定价偏差）
- 追求原则性与灵活性的动态平衡

**体现在实操上：**
- 三层漏斗分析 + 决策清单 = 理性框架（"守中"之"守"）
- 识别政策信号与市场情绪的阶段性机会 = 灵活应变（"守中"之"中"）

**典型提问：** "请用守中智慧帮我分析当前A股，既要看长期价值逻辑，也要看短期政策信号和情绪周期在哪里。"

### 8️⃣ 晶核五则（终极心法框架）

1. **问题定义是分析的第一性原理**：问对问题，解决一半
2. **"慢中快"三变量是理解复杂系统的手术刀**
3. **决策的最终产出必须是可执行的清单**
4. **"守中"智慧是体系落地的本地化编译器**
5. **复盘是系统迭代的核心燃料**——每次决策后都要复盘

## 模型组合使用建议

| 场景 | 组合链路 |
|------|---------|
| 全面分析一个行业/公司 | 慢中快 → 三层漏斗 → 决策清单 |
| 制定年度资产配置方案 | 生命周期模板 → 三维认知框架 → 守中智慧 |
| 判断某一投资机会 | 情境判断 → 三维认知框架 → 快中慢 → 决策清单 |
| 复盘投资决策 | 决策清单回顾 → 晶核五则逐条校验 |

## 数据源建议

- 宏观数据：国家统计局、央行报告、IMF/WB
- 产业数据：行业协会、券商研报、Wind/同花顺
- 公司数据：年报、季报、招股说明书
- 市场情绪：两融余额、换手率、新基金发行规模

## 提醒

- 以上框架提供的是思维结构，**不能替代专业的财务顾问意见**
- 投资有风险，所有决策请用户自行判断和承担
- AI可以辅助收集数据、构建模型、检查逻辑，但最终决策由用户负责
- 模型需要根据市场环境和个人情况灵活调整，不要机械套用
